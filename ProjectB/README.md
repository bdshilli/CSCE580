# Wetlands Prediction AI

## Setup
1. Put data into the folder data/processed_project/
2. Create a Python 3.12.12 environment
3. Run "pip install -r requirements.txt"
4. Run any model by typing "python MODEL_NAME.py"

## Data Notice
Please contact me at BDS7@email.sc.edu to retrieve the data to run the models. The individual file sizes are too large to be uploaded to github.
